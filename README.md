## Desarrollo

- Instalar dependencias
  
  ``` bash
  npm install
  ```

- Iniciar servidor
  
  ``` bash
  npm start
  ```

- Iniciar base de datos de desarrollo
  ``` bash
  docker-compose up
  ```

- Ejecutar tests
  
  ``` bash
  npm test
  ```
